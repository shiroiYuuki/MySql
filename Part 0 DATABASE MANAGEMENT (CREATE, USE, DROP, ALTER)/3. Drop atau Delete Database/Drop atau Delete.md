### 3. DROP DATABASE

---

Menghapus database secara permanen (termasuk seluruh tabel dan datanya).
⚠️HATI-HATI!⚠️ Setelah dihapus, data tidak bisa dikembalikan.

```sql
DROP DATABASE MyDB;
```

-- Hasil: Database MyDB dan seluruh isinya dihapus.
